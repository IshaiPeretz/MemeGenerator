# Meme Generator

A browser-based meme generator built with **HTML**, **CSS**, and **vanilla JavaScript**, using the **Canvas API** and a simple **MVC-style** architecture.  
Create custom memes directly in the browser and download them as images.

## Features

- Select a base image from a gallery (or a default set of images).
- Add multiple text lines (top, bottom, or anywhere on the canvas).
- Drag text on the canvas to reposition it.
- Change font size, alignment, and color.
- Render the final meme on an HTML5 canvas.
- Download the generated meme as an image file.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- HTML5 Canvas API
- MVC-style project structure (Model / View / Controller separation for practice)

## Learning Goals

This project was built to practice:

- Structuring a small app using an MVC-style pattern in plain JavaScript.
- Managing state in a Model (current meme, selected image, text lines).
- Separating rendering logic in the View from user interactions in the Controller.
- Working with the Canvas API for drawing images and text.
- Handling mouse events for selecting and dragging text on a canvas.
